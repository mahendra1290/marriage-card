## ADDED Requirements

### Requirement: Event Details
The application SHALL display the key wedding details clearly.

#### Scenario: Viewing Details
- **WHEN** the user navigates to the details section
- **THEN** they see:
    - Date and Time
    - Venue Name and Address (Churu, Rajasthan)
    - A "View on Map" button or embedded map

### Requirement: Map Integration
The venue section SHALL include a Google Maps integration.

#### Scenario: Interaction
- **WHEN** the user interacts with the map component
- **THEN** it opens the location in Google Maps app or shows an interactive embed
