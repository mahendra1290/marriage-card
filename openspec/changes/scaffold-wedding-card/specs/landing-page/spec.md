## ADDED Requirements

### Requirement: Hero Section
The landing page SHALL display a full-screen hero section heavily inspired by Studio Ghibli art style.

#### Scenario: Initial Load
- **WHEN** the user visits the home page
- **THEN** a lush, hand-painted style background is visible
- **AND** the title "Mahendra & [Bride Name]" fades in smoothly
- **AND** subtle parallax effects (e.g., moving clouds, swaying grass) are active

### Requirement: Animated Introduction
The page SHALL include a smooth entrance animation.

#### Scenario: Animation Sequence
- **WHEN** the page loads
- **THEN** elements animate in a staggered sequence (Background -> Title -> Date)
- **AND** the animation is smooth (target 60fps)
